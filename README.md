# Process-Track
A lightweight, single-file web app and Excel workbook for tracking institutional process improvement initiatives. Built around a navy-and-amber design system with a sidebar navigation, live dashboard, and project detail views.
# Features:
Project tracker with status, priority, progress, requirements, deliverables, and milestone tracking
Executive dashboard with live KPI cards and a stacked status bar — no manual updates
Issue & risk log linked to projects by ID
Add/remove projects via form; update status and % complete inline
Dropdown filters and live search across all projects
Dark mode support (HTML version)
Zero dependencies — pure HTML/CSS/JS in a single file; Excel version uses only built-in formulas

# Comes in two versions:
process_improvement_tracker.html — interactive web app, open in any browser
process_improvement_tracker.xlsx — Excel workbook with four sheets: Project Tracker, Dashboard, Issue Log, and Instructions
